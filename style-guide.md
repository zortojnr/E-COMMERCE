# TribesByWendy Logistics – Style Guide

## Brand Colors (Black + Gold)
Background Primary: #000000
Background Secondary: #0B0B0B
Accent Gradient: #D4A738 → #F5C76D
Text Primary: #FFFFFF
Text Muted: #E0E0E0
Border Accent: rgba(212,167,56,0.40)
Glow: rgba(245,199,109,0.25) / rgba(245,199,109,0.35) hover

Usage:
- Backgrounds: #000000 / #0B0B0B
- Text: #FFFFFF for primary, #E0E0E0 muted
- Accents: gold gradient bars, CTA fills, icon dots
- Borders: soft gold border accent

## Typography
Headings: Inter (bold)
Body/UI: Manrope

Scale:
- H1: 48/56
- H2: 32/40
- H3: 24/32
- Body: 16/24
- Caption: 13/20

## Components
- Navbar: fixed, transparent → solid on scroll, CTA “Book a Pickup”
- Hero: headline, subcopy, two CTAs (gold gradient + gold outline), badges, abstract route motif (gold)
- Service Tiles: 4 cards, gold top bar, soft gold borders
- Technology & Reliability: feature bullets with gold dots + KPI strip (CountUp)
- Customer Proof: teaser with gold CTA
- CTA Band: strong message + gold gradient primary CTA
- Footer: minimal corporate, contact and office info

## Patterns
- Abstract movement motif via inline SVG with gold gradients
- Gradient bars: linear-gradient(90deg, #D4A738, #F5C76D)
- Dots/indicators: small circles with gold fills
- Motion: subtle fade/translate; respect prefers-reduced-motion

## Accessibility & Performance
- Contrast AA+, focus rings on interactive elements
- Semantic landmarks: header, main, footer
- next/image for rasters; inline SVG for motifs
- Lazy-load noncritical blocks; minimal animation cost

## Responsive Breakpoints
- sm: 640, md: 768, lg: 1024, xl: 1280, 2xl: 1536

Layout Behavior:
- Hero: stack on mobile, two-column on desktop
- Tiles: 1 → 2 → 4 columns
- KPI: 2 → 4 cards equal width
- Footer: 1 → 2 columns

## Assets
- Logo: monochrome SVG (use existing mark in header)
- Icons: 24/48px SVG set for services
- Motif: RouteMotif inline SVG (gold gradient)

## CTAs
- Primary: “Book a Pickup”
- Secondary: “Contact Sales”

## Notes
- Keep copy concise and business-oriented
- Ensure navigational clarity and fast load

## Content Tone
Approved focus areas: supply chain management, transportation methods, warehousing operations, inventory control, distribution networks
Avoid: software, systems, automation, digital tools, platforms, APIs, tracking systems
Use concise, professional logistics language suitable for industry audiences

### Brand Line
- Tagline: “THINK LOGISTICS, THINK US”
- Placement: below logo in Navbar and visible in Footer
- Styling: text-xs, muted text color (#E0E0E0) on black background
